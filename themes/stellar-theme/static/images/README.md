# 背景图片说明

请将你的桌面壁纸复制到这个目录，并重命名为 `background.jpg`

## 操作步骤：

1. 找到你的桌面壁纸文件
2. 复制到这个目录：`C:\Users\steve\steven-blog\themes\stellar-theme\static\images\`
3. 重命名为 `background.jpg`

## 效果说明：

- **壁纸透明度**: 50%
- **蓝色覆盖层**: 70%透明度
- **蓝色颜色**: 更浅的天空蓝 (#f0f8ff)
- **文章卡片**: 85%透明度 + 毛玻璃效果

这样你可以同时看到壁纸和蓝色渐变，深色模式也能正常工作。
